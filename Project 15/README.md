# Customer Care Call Summarization

## Overview

The **Customer Care Call Summarization** application is a Streamlit-based web tool designed to streamline customer support operations. Users can upload multiple `.mp3` files of recorded customer care calls and automatically generate concise summaries, which can then be sent via email. This tool enhances efficiency by handling large volumes of customer interactions with ease.

## Key Features

- **Multi-File Upload**: Supports uploading multiple `.mp3` files for batch processing.
- **File Display Table**: Lists all uploaded files in a clear, tabular format for easy management.
- **Automated Email Summaries**: Generates and sends email summaries for each uploaded call with a single click.

## Technology Stack

- **Streamlit**: Provides the framework for creating an interactive and user-friendly web interface.
- **Python**: Core language for scripting and implementing functionalities.

## How to Use

### Access the Application

1. Open your web browser and navigate to [http://localhost:8501](http://localhost:8501) or the URL where the Streamlit app is hosted.

### Upload `.mp3` Files

2. Click the upload button to select and upload one or more `.mp3` files containing recorded customer care calls.

### Send Summaries via Email

3. After uploading, click the "Send Email" button next to each file in the table to send a summarized version of the call directly to your email.
