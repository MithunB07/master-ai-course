
# MASTER AI COURSE

Welcome to the ** MASTER AI COURSE ** repository! This collection features a variety of AI-powered applications, each built using top-notch technologies like LangChain, OpenAI, Hugging Face, and LLAMA2. From chatbots to data analysis tools, each app has something unique to offer.

## Applications Overview

### 1. Question & Answer App
Get quick answers based on pre-set data. This app uses advanced language models to provide accurate and relevant responses.

### 2. Conversational Chatbot
Engage in simple conversations with this basic chatbot. It's perfect for building interactive user experiences.

### 3. Kids' Similarity Finder
A fun, educational app that helps kids match similar items using AI. Learning through play has never been so easy!

### 4. Marketing Campaign Tool
Design and analyze marketing campaigns effortlessly. This app generates ideas, tracks trends, and offers insights to enhance your strategies.

### 5. ChatGPT Clone with Summarization
A ChatGPT clone with a twist—it summarizes long chats, making it easier to extract key points.

### 6. MCQ Quiz Maker
Create and manage multiple-choice quizzes with ease. Includes features for categorizing questions and automated scoring.

### 7. CSV Data Analysis Tool
Analyze CSV data with ease—clean, visualize, and understand your data in just a few clicks.

### 8. YouTube Script Writer
Need help scripting your next video? This app generates engaging, well-structured scripts based on your input topics.

### 9. Website Support Chat Bot
Enhance customer support with this chatbot, capable of handling FAQs and assisting users in real-time.

### 10. Automated Ticket Classification
Classify support tickets automatically, making it easier to manage and respond to customer queries.

### 11. Resume Screening Tool
Streamline your recruitment process with this AI tool that ranks resumes based on job requirements.

### 12. Email Generator using LLAMA2
Generate professional emails effortlessly. Just set your parameters, and let the app do the rest.

### 13. Invoice Extraction Bot
Automatically extract key details from invoices, saving time on data entry.

### 14. Text-to-SQL Query Helper
Turn natural language into SQL queries instantly—no SQL knowledge required.

### 15. Customer Care Call Summary
Summarize customer calls and generate alerts to track important points.

### 16. Code Review Assistant
Get AI-driven feedback on your code, focusing on quality and best practices.

## Installation

1. **Install Required Packages**:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. **Run the App**:
   ```bash
   streamlit run app.py
   ```

## Configuration

- **CSV File**: Ensure the CSV file is correctly formatted and in the right directory.
- **API Tokens**: Set up API tokens for Hugging Face models and Pinecone in the `.env` file.
- **Pinecone API Key**: Include the Pinecone API key in the `.env` file for data storage and retrieval.
