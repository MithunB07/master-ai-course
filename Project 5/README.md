# Chat GPT Clone with Streamlit

This repository contains a simple chat application built using Streamlit and HuggingFace models for conversational AI and summarization.

## Features

- **Chat with a language model**: Engage in conversation with an AI-powered model.
- **Summarize the conversation**: Generate concise summaries of the chat.

## Requirements

- Python 3.7 or higher
- Streamlit
- Langchain
- HuggingFace
- Langchain Community
- HuggingFace Hub
- Streamlit Chat

## Model Configuration

The application leverages two Hugging Face models:

- **google/pegasus-cnn_dailymail** for text summarization.
- **HuggingFaceH4/zephyr-7b-alpha** for conversational interactions.

Make sure you have access to these models through HuggingFace.
